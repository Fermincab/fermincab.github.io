This is a port of the A+ APL (4.22) programming language to IOS on IPad and IPhone. A+ was created by Arthur Whitney at the tier-1 invesment bank Morgan Stanley and is cousin to Q and K created by the same author and commericalized as KDB. KDB is used in high performance electronic trading systems around the world. Only requires IOS 5.1.1 or better. This is a console app; we recommend using "Prompt" SSH with a customized keyboard featuring common symbols like parenthesis.



A full on-device OCaml development environment to be released soon on the QMole desktop for Iphone and IPad.
