Full OCaml (3.12.0) distribution including ocamlc compiler, REPL and meta programming campl4. Only requires IOS 5.1.1 or better. Includes 900+ files and about 40MB - so install over Wi-FI is suggested. This is a console app; we recommend using "Prompt" SSH with a customized keyboard featuring common symbols like parenthesis.
