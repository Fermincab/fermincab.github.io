ANTLR (ANother Tool for Language Recognition) is a powerful parser generator for reading, processing, executing, or translating structured text or binary files. It's widely used to build languages, tools, and frameworks. From a grammar, ANTLR generates a parser that can build and walk parse trees. This is the distribution of ANTLR for iOS. This is a console app; we recommend using "Prompt" SSH with a customized keyboard featuring common symbols like parenthesis.
