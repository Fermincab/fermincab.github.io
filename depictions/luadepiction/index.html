This is a port of the popular functional scripting language Lua to iPhone and iPad. This release is aimed at developers: It comes complete with source headers, luadoc documentation, the Lua compiler as well as the Lua JIT compiler. Everything is included to create Lua applications on the iPhone or iPad directly.
