ZeroMQ (also spelled ZeroMQ, 0MQ or ZMQ) is a high-performance asynchronous messaging library aimed at use in scalable distributed or concurrent applications. This is a distribution of ZMQ for iOS. It is a library that ships with both static and dynamic libraries, header files and man page documentation.
