Full Gambit Scheme (4.6.4) distribution as created by Marc Feeley at the University of Montreal, ported to IOS; includes gsi REPL and gsc compiler. Only requires IOS 5.1.1 or better. This is a console app; we recommend using "Prompt" SSH with a customized keyboard featuring common symbols like parenthesis.



A full on-device Scheme development environment to be released soon on the QMole desktop for Iphone and IPad.
