Avian is a lightweight virtual machine designed to be both fast and small.
  1. Just-In-Time (JIT) compilation for fast method execution
  2. Generational, copying garbage collection ensures short pause times and good spatial locality
  3. Thread-local heaps provide O(1) memory allocation with no synchronization overhead
This is the iOS port of Avian built in both dynamic & static configurations, accompanied by its own libjvm library; installs into its own folder under /usr/local.
